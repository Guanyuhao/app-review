# App Store 上架资料（冷钱包）

本目录用于集中存放 **App Store Connect** 上架所需材料（截图、隐私政策、审核备注、合规模板）。

## 截图

- `iphone_67_1290x2796/`：iPhone 6.7 英寸常用尺寸（1290×2796）
- `iphone_65_1242x2688/`：iPhone 6.5 英寸常用尺寸（1242×2688）

> 建议：在 App Store Connect 里优先上传 6.7 英寸这一套；如仍要求 6.5，再补 6.5。

## 必填文档/文案（建议直接在 App Store Connect 粘贴）

- `PRIVACY_POLICY_TEMPLATE.md`：隐私政策模板（你们需要**发布到可访问的 HTTPS URL**）
- `APP_REVIEW_NOTES_TEMPLATE.md`：审核备注/测试步骤模板（重点避免 2.1 Completeness）
- `EXPORT_COMPLIANCE_TEMPLATE.md`：出口加密合规模板（回答 App Store Connect 的加密问题）
- `STORE_LISTING_COPY.md`：商店文案（名称/副标题/描述/关键词/支持信息）
